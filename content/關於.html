關於

內容管理系統以https://github.com/mdecourse/cmsimde 作為子模塊此 通道，可以指定對應的版本，cmsimde可以持續改版，不會影響內容子模塊，使用舊版cmsimde模塊的管理相關道。

利用cmsimde建立動態網誌方法：

1.在github建立網上，git clone到近端

2. 參考 https://github.com/mdecourse/newcms，加入除了 cmsimde 目錄外的所有內容

以 git 子模塊添加 https://github.com/mdecourse/cmsimde  cmsimde

建立cmsimde目錄，並從github取下子模組內容。

3.在近端維護時，從目錄到遠程中的cmsimde，以python wsgi.py啟動近端網際服務器。

動態內容編輯完成後，以generate_pages轉為內容內容，以git add commit及push將內容推到盡頭。

4.之後若要以git clone取下包含子模塊的所有內容，執行：

git clone --recurse-submodules https://github.com/mdecourse/newcms.git
